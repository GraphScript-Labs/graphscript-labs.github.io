# JS Repo Template

AttAditya's JS Repo Template is a boilerplate for JavaScript/TypeScript projects, providing a structured and organized starting point for developers. It includes a predefined directory structure, essential files, and scripts to streamline the development process.

[![Sponsor](https://img.shields.io/badge/Sponsor%20PROJECT-db61a2?style=for-the-badge&logo=github)](https://github.com/sponsors/AttAditya)

</div>

## Setup Checklist

- [ ] `config.sh`: set the variables about the development environment
- [ ] `package.json`: swap `<repo>` and `<description>` with the name and description of the repository
- [ ] `README.md`: swap `<repo>` with the name of the repository and update the description and other relevant sections
- [ ] Add secrets and variables in the GitHub Repository:
  - `secrets.AI_URL`
  - `secrets.AI_TOKEN`

## Installation

```sh
# Clone the repository
git clone https://github.com/AttAditya/<repo>.git
cd <repo>

# Source the development environment
source dev.sh

# Setup and run
dev setup
dev run
```

## About

This project is designed to facilitate the development of JavaScript applications with a focus on modularity and scalability. It provides a clear separation of concerns, allowing developers to easily manage different components of their application.
